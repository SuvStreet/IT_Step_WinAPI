Кодировки
===

Кодировки в Windows
---

1. ANSI
* 1 байт на символ
* мультибайтовая (многобайтовая) кодировка
* позволяет работать с латиницей
* тип **`char`**

2. Unicode
* 2 байта на символ
* позволяет работать с символами разных языков мира
* тип **`wchar_t`**
* перед строкой нужен префикс **`L`**
